# 데이터베이스 스키마 확인

.claude/docs/DATABASE.md 파일을 읽고 사용자가 요청한 테이블이나 쿼리 패턴을 찾아 설명하세요.

만약 사용자가 특정 테이블을 요청하지 않았다면:
- 주요 테이블 목록 (users, construction_management, site_summary, weekly_plans, daily_plans, sales_activities, invoice_records)
- 각 테이블의 용도를 간단히 요약

만약 특정 테이블을 요청했다면:
- 테이블 구조 (컬럼, 타입, 용도)
- 관련 쿼리 패턴
- 다른 테이블과의 관계
